# Electric-Vehicle-Sales-Analysis
Power BI dashboard analyzing Electric Vehicle (EV) sales trends in India with category-wise insights, state-wise analysis, and interactive visualizations.
Electric Vehicle Sales Analysis & Dashboard

Project Overview

This project analyzes Electric Vehicle (EV) sales data in India using Power BI. The dashboard provides insights into sales trends, vehicle categories, and regional performance.

---

Objectives

* Analyze EV sales trends (2015–2020)
* Identify top-performing states
* Compare vehicle categories (2W, 3W, 4W)
* Build an interactive dashboard

---

 Tools & Technologies

* Power BI
* Python (Pandas, NumPy)
* SQL
* Excel

---

Dashboard Features

* KPI Cards (Total Sales: **3.6M**)
* Category-wise analysis
* State-wise EV sales
* Vehicle type breakdown
* Year & category filters

 Key Insights

* 2-Wheelers dominate (~50%)
* 3-Wheelers contribute (~45%)
* 4-Wheelers have growth potential
* Top states: Uttar Pradesh, Maharashtra, Karnataka

--- Dataset

Contains EV sales data categorized by:

* Vehicle Category
* Vehicle Type
* State
* Year
